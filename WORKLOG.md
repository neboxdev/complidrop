# Worklog

Free-form narrative log of CompliDrop sessions — what changed, why, and what's stuck. Structured records live in commits, tickets, and ADRs; this file is the human-readable thread between them.

Use `/worklog` to add a new entry. (Automated per-session markers go to the gitignored `.claude/session-log.md`, not here — this file stays clean for intentional entries.)
